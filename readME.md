#### 특수정렬
    정렬할 값들이 단순할 경우, 이 단순함을 이용해서 O(n) 시간에 정렬할 수 있는 경우가 있다.
    즉 정렬할 값들이 특수한 성질을 만족할 경우, 그 성질을 이용해서 O(n) 시간에 정렬하는 알고리즘들을 
    특수 정렬 알고리즘이라고 한다.

    * 계수 정렬 (countingSort)
    * 기수 정렬 (radix sort)
